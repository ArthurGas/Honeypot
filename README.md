# Honeypot
Лабораторная по созданию сетевой приманки имитирующей машину с открытыми портами 21, 22, 23, 25 и 80, соответствующие службам FTP, SSH, Telnet, SMTP и HTTP.  
Работает для linux и windows  
требования программы:  
    • установленный интерпретатор языка python 3 версии;  
    • модуль socket для python3;  
    • модуль threading для python3;  
    • модуль logging для python3;  
    • модуль datetime для python3.  
Логгирование в файл honeypot.log той же директории.
